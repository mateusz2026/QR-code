# QR Code Component

## 📌 Overview
This project is a solution to the Frontend Mentor QR Code component challenge.  
The goal was to build a clean and responsive card layout using HTML and CSS.

## 🚀 Live Demo
https://mateusz2026.github.io/Frontend-Mentor-QR-code/

## 🛠️ Built with
- HTML5
- CSS3
- Flexbox
- Mobile-first workflow

## 📱 Features
- Responsive design (mobile & desktop)
- Centered layout using Flexbox
- Clean and simple UI

## 📚 What I learned
- How to center elements using Flexbox
- How to structure HTML properly
- How to work with spacing and typography
